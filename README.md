# Riviera Sidecar — Site Web

Site officiel de **Riviera Sidecar**, Nice — French Riviera.

## Déploiement

Déposer `index.html` à la racine du dépôt GitHub.  
Activer **GitHub Pages** → Settings → Pages → Branch: `main` → `/root`.

## Fonctionnalités

- Design vintage Art Déco (1920s–1950s)
- 5 langues : Français, English, Deutsch, Italiano, Español
- 100% responsive (mobile / tablette / desktop)
- Aucune dépendance externe — fichier unique auto-suffisant
